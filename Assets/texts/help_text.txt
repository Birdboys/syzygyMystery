[b]COMMANDS[/b]
[b]Map:[/b] Open map
[b]Go:[/b] Move to an adjacent room
[b]Look:[/b] Pull up photo and description of current room, object in current room, or object in inventory
[b]Take:[/b] Attempt to take object and put it in inventory
[b]Use:[/b] Attempt to use object in current room or inventory
[b]Speak:[/b] Speak to person in current room
[b]Inventory:[/b] Open inventory
[b]Help:[/b] Print list of commands and their functions
